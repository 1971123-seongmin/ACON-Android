## 기능
| 기능 | 설명 |
|------|------|
| **회원가입/로그인** | 구글 소셜 로그인 지원 |
| **온보딩** | 개인 맞춤형 음식 취향 설정 |
| **지역 인증** | 현재 위치 기반 지역 인증 |
| **맛집 추천** | 위치 기반 식당/카페 추천, 다양한 필터 옵션 |
| **맛집 상세보기** | 사진/메뉴/메뉴판 제공, 지도 앱 길찾기, 북마크, 딥링크로 식당 공유 |
| **리뷰 등록** | 도토리로 리뷰 작성 |
| **맛집 업로드** | 사용자가 맛집 등록 (사진, 메뉴 포함) |
| **프로필** | 프로필 수정 및 조회 |
| **북마크** | 저장한 맛집 모아보기 |
